File Encryption WinForms

Created By: Marcus Schuff dba Schuff Software Company

Created: March, 2020

Version: 1.0.0.3

About:

File Encryption WinForms is a simple tool to use Microsoft's built-in methods to encrypt your files using your Windows user account.

Limitations:

Windows Pro+ Only.

Will not work with Windows Home versions. See the explanation from Microsoft below.

This API is only supported on Windows platforms that are able to use the NTFS Encrypting File System (EFS). Any attempt to use this on non-Windows systems, Windows Home Edition systems, or non-NTFS drives results in a PlatformNotSupportedException or NotSupportedException, depending on the situation.
